# 坦克大战

一个经典的HTML5坦克大战游戏，使用纯JavaScript实现。

## 游戏特性

- 经典坦克大战玩法
- 玩家坦克控制（方向键移动，空格键射击）
- 敌人AI自动寻路和射击
- 砖墙、钢墙、水域等地形元素
- 基地保护机制
- 多关卡系统
- 得分和生命系统

## 操作说明

- **方向键**：移动坦克
- **空格键**：发射炮弹

## 游戏规则

1. 保护你的基地（粉色方块）不被摧毁
2. 消灭所有敌人坦克进入下一关
3. 你有3条生命，被敌人击中会损失一条生命
4. 消灭敌人获得100分

## 在线游玩

直接在浏览器中打开 `index.html` 文件即可开始游戏。

## 技术栈

- HTML5 Canvas
- 原生JavaScript
- 无需任何外部依赖
